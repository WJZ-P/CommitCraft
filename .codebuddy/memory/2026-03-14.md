## 修复：Cloudflare Workers Error 1101（next-intl + vinext 兼容性）

**问题**：项目部署到 Cloudflare Workers 后页面打不开，报 Error 1101。
**根因**：`next-intl/plugin` 的 `createNextIntlPlugin()` 是一个 Webpack 插件，它设置 `next-intl/config` → `./i18n/request.ts` 的 alias。vinext 使用 Vite 构建，不执行 Webpack 配置（构建日志确认：`[vinext] next.config option "webpack" is not yet supported and will be ignored`），导致运行时 `next-intl` 找不到 i18n 配置文件而崩溃。
**修复**：在 `vite.config.ts` 中添加 `resolve.alias: { "next-intl/config": path.resolve(__dirname, "./i18n/request.ts") }`，手动实现同样的别名映射。
